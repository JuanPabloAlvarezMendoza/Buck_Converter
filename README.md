En este repositorio encontrará toda la información relacionada con el proyecto desarrollado. Este proyecto consiste en el diseño y análisis de un buck converter, con el objetivo de comprender su funcionamiento, su proceso de diseño y los criterios eléctricos involucrados. Además, el proyecto busca fortalecer nuestras habilidades en la fabricación de PCB, así como adquirir experiencia en su envío, manejo y posibles complicaciones logísticas.

Agradecemos su atención y colaboración.
